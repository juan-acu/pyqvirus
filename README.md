# PyQVirus: Quantum Key Distribution Simulations

This repository contains Python simulations of quantum key distribution (QKD) protocols, specifically:

- **BB84** protocol
- **E91** protocol

These scripts allow users to experiment with secure quantum communication, including scenarios with and without eavesdropping.
